<blockquote>有些人25岁就“死了”，只是到75岁才“埋葬”</blockquote>

<img src="http://img.full-stacker.com/%E6%9E%B7%E9%94%811.jpg" alt="" />

前几天在朋友圈发了张二维码，使我的小公众账号涨粉高达个位数，也是没shei了。但是有个朋友们十分关心的问题我还是挺想回答一下的，如题，我为啥要搞这么个公众账号。

其实早就申请了这么个公众账号了，应该是long long ago的2013年了吧。当时是一时兴起，写了个公众账号的程序，最终也没能实现什么吸引人的功能，一个简单的回复，一个查询天气预报，然后就不了了之了。。。直到最近看到各大头条报道公众账号盈利n千万，我的心深深被震撼了，在后台看了下我两位数的粉丝量，这新闻当然不是我开始写点东西的原因了。

好吧，言归正传。突然想写点东西，其实就是为了给自己找点乐子。不至于使生活太程序化。还记得那句话，<a href="http://mp.weixin.qq.com/s?src=3&amp;timestamp=1480416908&amp;ver=1&amp;signature=eiF2xsksBcgi9JHxgXbpCm-jJyJsjsjn4Zgcqa4AW2gJ15yOCN2GjO7OEYDUkCtsf-pXySpnd14aVndUp6sb*XJ8BUvFD2l7jiv*EOtAXazqvmqPUCk1Gm-OzG1qAVnTLki6vK8PUskNzrZk7cxNCQ==">有些人25岁就“死了”，只是到75岁才“埋葬”</a>。最近几个月对引言的上半部分有点感触吧，这不是笑话，今年我二十五周岁！为什么感觉像“死了"呢？先郑重的以文字形式吐槽下今年的加班生活吧，一句话以蔽之，除了九月份双休，其他时间定期或不定期加班。为什么有的定期有的不定期呢，因为上一家公司加班不定期，现在这家定期，所以说程序猿，你还以为换工作能逃脱加班的命运吗。倒不是说拒绝加班，IT dog 们都感受过来自工作量，上线时间，项目进度，dead line 压力，不加班怎么去完成任务呢。但一遍遍的复制粘贴，一次次的重复，没有丝毫新知识，新鲜事物来刺激下大脑，把激情一点点的都耗光了。所以就像机器一样，今天重复昨天，明天重复着今天。是搬砖还是在创造？所以说呢，写点东西，算是给自己个小小的挑战，也给自己跳出一天天的重复找个突破口。

<hr />

本来想把第一篇文章写的长一点，漂亮点，但写了这么两段话，突然感觉写不下去了。那么咱们扯点别的吧。

要不谈谈理想？

刚刚读完一本书《财务自由之路》，一本关于财富的书。书中讲到了如何赚到第一个100万，如何走向财富自由的三个阶段：财务保障，财富安全，财务自由。但在实现这些之前，你需要明确你真正想要什么，你的目标是什么？

暂停下，想想你还有目标吗？

我想了好久，最后得出的结论是：先攒够了买房的钱，其他的事情慢慢来。之所以其他的事情要慢慢来，是因为其他的事情想不清了。想不清还要去做哪些事。当时吹牛x,说下要一起做改变世界的程序员的“誓言”，现在却感觉离改变世界越走越远。经过三年的工作，发现同学们都开始重新定位和思考自己的职业之路，做出新的选择。想了想好像除了code code and code 啥都不会了，定位到哪依然是个想不清的问题。

为了不将负面情绪表达出来，就不再往下写了。

快速结束语：在想清楚要干什么之前，我的梦想依然是做个改变世界的程序员！
